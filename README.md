# ENDLESS Website (Working Buttons Version)

This version has working page links.

## Pages
- `index.html` — homepage
- `shop.html` — shop page
- `product-v4.html` — product page
- `contact.html` — contact / drop list page
- `styles.css` — shared styling
- `assets/` — images

## How to upload to GitHub
1. Create a new GitHub repository
2. Upload all files from this folder
3. Keep the structure exactly the same
4. Go to Settings -> Pages
5. Set Source to Deploy from branch
6. Choose `main` and `/root`
7. Save

## Replace later
You can later replace:
- `shop.html` buttons with real checkout links
- `contact.html` email with your real email
- nav text and footer text with your real brand info
